# docker-ti-ccs-msp430

Docker image with TI Code Composer Studio and MSP430 libraries installed. 

> [!NOTE]
> Using CCS v20.2.0.00012 and Ubuntu 22.04 as base image.

## Build

```bash
docker build . -t freitasrenato/ti-ccs-msp430
```
